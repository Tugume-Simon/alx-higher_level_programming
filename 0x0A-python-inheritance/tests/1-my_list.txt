The ``1-my_list`` module
========================

Using ``MyList`` class
----------------------

This file tests a module 1-my_list that
inherrits from the list class

Import:
	>>> MyList = __import__('1-my_list').MyList
	>>> list1 = MyList([7, 4, 5])

Tests:
	>>> MyList()
	[]
	
	>>> MyList([1, 2, 3])
	[1, 2, 3]

	>>> list1 + [3, 8, 1]
	[7, 4, 5, 3, 8, 1]

	>>> list1
	[7, 4, 5]

	>>> list1 += [3, 8, 1]
	>>> list1.print_sorted()
	[1, 3, 4, 5, 7, 8]

	>>> list1.append(2)
	>>> list1
	[7, 4, 5, 3, 8, 1, 2]

	>>> list1[3]
	3

	>>> list1.sort()
	>>> list1.pop()
	8
